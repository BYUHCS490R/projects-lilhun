HTML attributes css is an attribute that is in the html element. This is not part of html anymore but browsers can still recognize this. Inline css is applying style inside the html tag. Embedded css is applying css using style tags on the html page. External css, the best method to use is to link a seperate css stylesheet the the html page.

It is best to only use external css becasuse it is easy to have all the css in one place and you don't have to wory about certaion styles being overwritten.

A selector is used to apply certain css to a specific html element.
