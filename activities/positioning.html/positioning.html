Normal flow is where elements are below eachother and take up the width of the page.

float allows text to wrap around images but the element is taken out of the normal document flow. To avoid layout issues it is best not to use float.

aboulute, fixed and realive are attributes of the position property. Relative keeps an element in the normal flow but allows some positioning. In absolute the element is removed from the normal flow and can be positioned whereever you want. In fixed the image is removed from normal flow but always stay relative the the users veiwpoint.
