structural tags

header, nav,main,footer

The header tag contains the title and logo of the website, it also can contain nav tags.

The nav tag contains link to diffrent pages in you site not pages on other sites. It is usually in the header because it is convinient. In can be included elsewhere but that is uncommon.

The main tag contains all the code for the main part of the page

the footer can contain contact info, copyright info, and links to other less common but helpful pages.
