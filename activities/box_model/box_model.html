padding margin border

block tags are elements that take up the entire width of the page in normal flow. Inline elements fall inline with everything else and do not take up the width of the page or cause line breaks.

margin is the area around a block element. Padding controls how close the stuff inside the block is relative to the margin.a border is around the outer edge of the block element.

You can use pixels, percentages, em, and rem

a figure tag can be used to group an image and a caption together.
