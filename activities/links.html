The root is the directory holds all the other folders in the website. Domain name specifies the computer,and the path specifies the files and folders it needs to go through to find a certain page.

A default document is a file named index.html for each folder.

An apsolute link goes to a page in a different website while a relative link goes to a page in the same website.

Relative links rules: If you want to go to a diffrent folder type foldername/file. Use .. to jump up a folder. start over = / at the beginning.
